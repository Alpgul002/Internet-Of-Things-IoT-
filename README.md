# Internet of Things Projects

This repository contains a collection of projects and assignments developed in the field of **Internet of Things (IoT)**. The projects focus on collecting, processing, analyzing, and interpreting data from different sources such as sensors, Raspberry Pi devices, local computers, and online databases.

The main purpose of this repository is to demonstrate how IoT systems can be combined with data analysis and machine learning techniques to extract meaningful information from real-world or externally collected data.

## Project Overview

The repository includes different IoT-related works involving:

* Sensor data collection and processing
* Machine learning applications
* Classification and regression models
* Deep learning experiments
* Raspberry Pi-based computation
* Local and remote computation
* Wi-Fi fingerprinting
* Data gathered from online databases
* Data analysis using Python notebooks

Some parts of the work were implemented and tested using a **Raspberry Pi**, while others were developed and executed on a personal computer. The projects include both hardware-related IoT concepts and software-based data analysis workflows.

## Topics Covered

### Machine Learning

Several projects in this repository use machine learning methods to analyze data. These include classification and regression tasks, where models are trained to make predictions based on given datasets.

Implemented methods include:

* K-Nearest Neighbors
* Linear Regression
* Classification workflows
* Data preprocessing
* Model evaluation

### Deep Learning

The repository also includes deep learning experiments using neural network-based models. These projects focus on training and testing models on collected or prepared datasets.

Covered topics include:

* Convolutional Neural Networks
* Model training
* Dataset preparation
* Requirement-based environment setup

### IoT and Sensor-Based Data

Some projects are related to sensor data and IoT devices. The data used in these projects comes from physical devices, Raspberry Pi-based setups, local computation, remote computation, or online data sources.

The goal is to show how sensor or IoT-related data can be collected, transferred, processed, and analyzed.

### Raspberry Pi and Remote Computation

Part of the work involves Raspberry Pi and remote computation concepts. These projects explore how computation can be performed locally or remotely depending on the hardware, network, and system requirements.

### Wi-Fi Fingerprinting

The repository also includes work related to Wi-Fi fingerprinting, which is a technique used for location estimation based on wireless signal patterns.

## How to Use

Clone the repository:

```bash
git clone https://github.com/your-username/your-repository-name.git
```

Go into the project directory:

```bash
cd your-repository-name
```

Open the notebooks using Jupyter Notebook or JupyterLab:

```bash
jupyter notebook
```

Install required libraries if a requirements file is provided:

```bash
pip install -r requirements.txt
```

Then open and run the notebook files step by step.
